# 🛒 E-Commerce Website

Responsive e-commerce frontend built using HTML, Tailwind CSS, and JavaScript.

## 🚀 Features
- Product listing
- Search
- Filtering
- Wishlist
- Cart
- Login / Logout
- Checkout
- Invoice
- Responsive Design

## 🛠️ Tech Stack
- HTML
- Tailwind CSS
- JavaScript
- Axios
