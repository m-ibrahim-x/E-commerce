# 🛒 E-Commerce Website

A modern and responsive e-commerce frontend built using **HTML, Tailwind CSS, and JavaScript**.
The project simulates a real online store with product listing, filtering, and user interactions.

---

## 🚀 Features

* 🛍️ Display products dynamically using JSON
* 🔍 Filter products by category
* ❤️ Add to Wishlist
* 🛒 Add / Remove from Cart
* 💾 LocalStorage support (save user data)
* 📱 Fully responsive design
* 🎨 Modern UI with smooth animations
* 📂 Sidebar navigation system

---

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS
* JavaScript (ES6)
* Axios
* LocalStorage
